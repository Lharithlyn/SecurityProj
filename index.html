Introduction:

This project introduces the threats of Cross-Site Scripting in the context of information security.

This project explores website vulnerabilities that may be exploited by Cross-Site Scripting and preventative measures against these exploits.

The main component of this project features twin websites, with one vulnerabile to attacks, and one with security measures in place.


Project Overview:

The dual websites take the form of a mock class website on information security. These fake websites serve the purpose of educating about the dangers of cross site scripting and the vulnerabiles and information that may be stolen if no preventative measures are in place. Though both the secure and insecure website take on the same appearance, They are different in terms of backend logic and how user input is handled.
