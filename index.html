Introduction:

This project introduces the threats of Cross-Site Scripting in the context of information security. This project explores website vulnerabilities that may be exploited by Cross-Site Scripting and preventative measures against these exploits. The main component of this project features twin websites, with one vulnerabile to attacks, and one with security measures in place.
